# 📡 PFE — Monitoring Antennes Tunisie Telecom (Mahdia)

## 🎯 Objectif

Simulation et visualisation des antennes réseau dans le gouvernorat de Mahdia avec analyse des performances.

---

## 🧱 Technologies utilisées

* PostgreSQL + PostGIS
* GeoServer (WMS/WFS)
* GeoNetwork
* Grafana
* Python (simulation)

---

## 🚀 Lancer le projet

```bash
docker-compose up -d
```

```bash
python data/simulation.py
```

---

## 🗺️ Accès

* GeoServer: http://localhost:8081/geoserver
* GeoNetwork: http://localhost:8080/geonetwork
* Grafana: http://localhost:3000

---

## 📊 Données

Chaque antenne contient:

* 📍 Position (lat/lon)
* 🌡️ Température
* ⚙️ CPU
* 🚨 Statut (normal / alerte / critique)
